Match the pairs

About the game

Nothing fixes a thing so intensely in the memory as the wish to forget it. -Michel de Montaigne
Memory plays a crucial in a person’s success.so, test your memory skills by playing our ‘Match the pairs’. The crux of this game lies in matching pairs of a symbol in a single try. If you have guessed wrong, then the symbols disappear and will have to be committed into your memory ! Have fun in finding out the pairs !
